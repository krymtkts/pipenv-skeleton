Pipenv simple skeleton
======================

A simple skeleton of pipenv project.

This project includes settings below.

- Python 3.7
- autopep8
- Pylint
- YAPF
    - Google style format
- mypy

Usage
-----

Clone or copy this project.

### Commands

```sh
# Install modules.
pipenv install --dev

# Run mypy with strict mode.
pipenv run mypys
```
